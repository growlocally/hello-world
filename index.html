03 — Approach

A point of view, not a process diagram.

"The best web work isn't a project — it's an operating system. Tokens, components, content models, and motion that compound across every campaign, region, and quarter." D. Cho — design notes, 2025
  1. A·01

    Systems before screens.

    If it can't be assembled twice, it isn't a design — it's a one-off.

  2. A·02

    Strategy is a deliverable.

    A decision doc beats a Figma frame when the question is "why ship this."

  3. A·03

    Motion is structural.

    Used well, it teaches the page. Used badly, it taxes the user.

  4. A·04

    Measure the thing.

    If we shipped it, we instrumented it. Otherwise we shipped a hope.
